# 响应式系统的依赖收集追踪原理

## 为什么要依赖收集

举个例子，现在有这么一个 Vue 对象

```javascript
new Vue({
  template: `<div>
      <span>{{text1}}</span>
      <span>{{text2}}</span>
    </div>`,
  data: {
    text1: '文本1',
    text2: '文本2',
    text3: '文本3'
  }
})

// 修改 text3
this.text3 = '文本3-改'
```

修改了 `data` 中的 `text3` 的数据，但是因为视图中并不需要 `text3`，所以并不需要触发前面所写的 `updateView` 来更新视图，调用 `updateView` 显然是不正确的

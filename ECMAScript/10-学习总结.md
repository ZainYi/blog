# 学习总结

- 看完《学习 JavaScript 数据结构与算法》这本书，了解了一些基本的数据结构和算法方面的思想
- 学习数据结构和算法最重要的是学习其中的思想和思维逻辑，不同编程语言有不同的实现方式，但体现的思想和解决的问题大致雷同
- 数据结构是生活中常见事件模型的抽象。在学习时，在对 **图** 那块的遍历还未理解透彻，需要找资料继续学习理解
- 算法板块对于 **归并排序**、**二分排序**、**堆排序** 的理解还未透彻，需要继续学习和理解
- 复习大学时代的课程

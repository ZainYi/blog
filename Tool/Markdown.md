# Markdown 语法

什么是 Markdown?

简单来说，Markdown 就是赋予了普通文本文件格式化的功能，同时又保持了文本文件的简洁

Markdown 简单易用，github 上的项目中的 README 默认使用 Markdown，以下是其语法

## 分级标题

语法

```md
# 一级标题

## 二级标题

### 三级标题

#### 四级标题

##### 五级标题

###### 六级标题
```
